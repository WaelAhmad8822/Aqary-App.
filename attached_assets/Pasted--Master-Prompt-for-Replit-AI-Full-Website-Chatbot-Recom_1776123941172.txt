### Implementation Requirements

**1. User Interaction Tracking**
- Track EVERY user action on property cards:
  - `view` (when card is visible)
  - `click` (when user clicks property)
  - `save` (when user clicks heart/favorite)
  - `contact` (when user clicks contact button)
  - `scroll` (when user scrolls past property)
  - `time_spent` (seconds spent viewing property details)
- Store in `Interactions` collection with weight values.

**2. Scoring Algorithm**
```javascript
Final Score = (Content_Score Г— 0.6) + (Behavior_Score Г— 0.4)

Content_Score = Cosine Similarity between:
  - User Preferences Vector [budget, location, type, features]
  - Property Vector [price, location, type, features]

Behavior_Score = Sum of all interaction weights for that property
  - view (1) + save (3) + contact (5) + scroll (0.5) + time (0.1/sec)
```

**3. Recommendation Display**
- Show top 10 properties on Home Page.
- Each property card must show:
  - Property image, title, price, location
  - **Match Reason** (e.g., "ЩҠЩҶШ§ШіШЁ Щ…ЩҠШІШ§ЩҶЩҠШӘЩғ"ШҢ "ЩҒЩҠ Щ…ЩҲЩӮШ№Щғ Ш§Щ„Щ…ЩҒШ¶Щ„")
  - Recommendation Score (hidden or shown as "Match %")
- Sort by highest score first.

**4. Continuous Learning**
- Every interaction updates user profile.
- Recommendations improve after each session.
- Store interaction history for 90 days (auto-delete for privacy).
